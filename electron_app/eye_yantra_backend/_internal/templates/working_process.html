Diagnostic Workflow

Complete step-by-step guide for professional eye diagnostic testing with automated precision and comprehensive reporting

👤
1

Enter Subject's Name

Begin by entering the name of the patient or subject. This helps organize and tag diagnostic data accurately for proper record keeping and identification.

🔍
2

Search Bluetooth Device

Turn on your diagnostic device and search for it using the built-in Bluetooth scanner. Ensure your device is in pairing mode for successful detection.

🔗
3

Connect with Device

Once your device appears in the list, click to connect. A secure Bluetooth connection enables real-time data synchronization with the application.

📶
4

Enter Wi-Fi Credentials

Provide the Wi-Fi network name and password to enable cloud-based functionalities including report storage, system updates, and data backup.

📹
5

Live Camera Preview

After successful setup, view the live feed from the subject-facing camera. Adjust positioning and ensure proper alignment for optimal test precision.

🧪
6

Select and Perform Tests

Choose from three available diagnostic tests. Each test is optimized for accuracy with automated guidance and real-time feedback systems.

✅
7

Complete All Tests

Ensure all three diagnostic tests are completed successfully. The system automatically verifies and stores results with timestamp data.

📊
8

Generate Report

Click "Generate Report" to compile test results into a structured diagnostic summary with analysis, recommendations, and visual data.

⬇️
9

Download Report

Hit the "Download" button to save the comprehensive report as PDF, including analysis results, image snapshots, and diagnostic markers.

Available Diagnostic Tests

Preliminary Eye Check

Basic visual assessment and eye health screening

Hirschberg Test

Corneal light reflex test for eye alignment evaluation

9-Gaze Test

Comprehensive eye movement and muscle function analysis

⚠️ Important: All tests must be completed for the proper report generation

💡 Professional Tips

Use optimal lighting conditions for camera-based tests to ensure accurate readings
Keep the subject still and focused during the 9-gaze test for precise measurements
Always double-check device connections before starting any diagnostic procedure
